Dmacc Attendance is a GUI based attendance management program written in Java using the Eclipse IDE for the Data Structures final project. Instructor add student to custom named classes. Attendance can then be taken for each class. Attendance reports can be generated for each class. An import feature allow list of student to be easily imported.

Where's my stuff is a web based personal lending library written in Java using the Spring platform and the Eclipse IDE. It was the final project for Java II. This project was written in a group which consisted of five members. My contribution to the project was to design and write the Html and Css, however each person added too the html as each new feature was implemented. Additionally, I implemented the lend item function, and I added the ability to upload pictures of items, which were stored in a database. Users can add items which can be lent out to others users for a predetermined amount of time. The web site keeps track of the items for each user, items lent out, lend dates, due dates, and borrower ratings

Rental Management system is a text based tenant management system for land lords. It was written in C++ using the QT IDE. The software allows landlords to enter building and tenants information and to associate a tenant with a building. Both tenants and the landlord can sent messages to individual users on the system. Additionally, Tenants can send maintenance requests and notice of rent payments to the landlord.
