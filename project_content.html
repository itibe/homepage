Dmacc Attendance is a GUI based attendance management program written in Java using the Eclipse IDE for Data Structures final project. Instructor add student to custom named class rooms. Attendance can then be taken for each class. Attendance reports can be generated for each class. An import feature allow list of student to be easily imported.

Where's my stuff is a web based personal lending library written in Java using the Spring Platform and the Eclipse IDE. This project was written in a group of five members. My contribution to the project was to design and write the HTML and CSS, however each person added too the HTML and CSS as each new feature was implemented. Additionally, I implemented the lend item function, and I added the ability to upload pictures, which were stored in a database. Users can add items which can be lent out to others users for a predetermined amount of time. The web site keeps track of the items for each user, items lent out, lend dates, due dates, and borrower ratings

The Stock Portfolio Organizer is a GUI based program written in Python using The JetBrains PyCharm IDE. The software will allow the user to add buy and sell transactions of any valid trading stock. The software will calculate the value of the current portfolio using real time stocks quotes from an api provided by Alphavantage.co. Profit and loss can also be calculated for each stock.

Rental Management system is a text based tenant management system for land lords. It was written in C++ using the QT IDE. The software allows landlords to enter building and tenants information and to associate a tenant with a building. Both tenants and the landlord can sent messages to individual users on the system. Additionally, Tenants can send maintenance requests and notice of rent payments to the landlord.
